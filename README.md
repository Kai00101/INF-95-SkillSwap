# SkillSwap

SkillSwap is a student-focused platform for exchanging skills and knowledge.  
Students can teach what they know and learn what they need from peers.

## Target Users
Primary: university students  
Future expansion: young professionals, self-learners, hobby learners, and skill mentors

## Features
- Skill exchange between students
- User profiles
- Skill listings
- Messaging / requests
- Firebase authentication and database
